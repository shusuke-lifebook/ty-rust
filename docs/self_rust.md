# 独習Rust
- Rust(ラスト)は、Mozillaにおける開発から始まったマルチパラダイムのプログラム言語である。
- 従来のプログラミング言語に似ているようで似ていない独自の世界を持つプログラミング言語がRustである。

| プログラム | 概要                               |
| :--------- | :--------------------------------- |
| rustup     | ツールチェインを管理               |
| rustc      | コンパイラー                       |
| cargo      | パッケージマネジャー(ビルドツール) |
| clippy     | リンター(構文解析)                 |
| rustfmt    | コードフォーマッター               |

## 新規のパッケージを作成する
- cargo new --bin intro
  - --binオプションは**パッケージクレート**であることを指定する
- コマンドによるビルド実行
  - cargo run --bin sample

## アイテム
| アイテム         | 概要                     |
| :--------------- | :----------------------- |
| mod { ... }      | モジュールの宣言         |
| extern crate ... | 外部クレートの参照       |
| use ...          | use宣言                  |
| fn ...           | 関数定義                 |
| type ...         | 型エイリアス             |
| struct ...       | 構造体                   |
| enum ...         | 列挙体                   |
| union ...        | 共用体                   |
| const ...        | 定数アイテム             |
| static ...       | 静的アイテム             |
| trait ....       | トレイト                 |
| impl             | メソッドやトレイトの実装 |
| extern { ... }   | 外部ブロック             |
| < ... >          | ジュネリック引数         |
| ...              | 関連アイテム             |
